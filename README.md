# Amazon-Frontend-Clone
This project is a frontend-only clone of the Amazon website , built purely using HTML and CSS . It replicates the visual design and layout of Amazon's homepage.

🚀 Features
🏠 Amazon-style homepage layout
🔍 Navigation bar with search box
📦 Product sections with images and pricing
🎨 Clean and structured CSS styling

🛠️ Tech Stack
HTML5
CSS3
